There is a lot of room for microtiming between and beyond shuffle (`|x-x|`) and swing:

| name    | pattern      | pattern                      | relation
|---------|--------------|------------------------------|-------------
|         | `|x----x--|` | `|x--------------x--------|` | 3/5 = 0.6
| shuffle | `|x-x|`      | `|x---------------x-------|` | 1/2 = 0.5
|         |              | `|x----------------x------|` | 7/17 ≈ 0.41
| swing   | `|x--x|`     | `|x-----------------x-----|` | 1/3 ≈ 0.33
|         |              | `|x------------------x----|` | 5/19 ≈ 0.26
|         | `|x---x|`    | `|x-------------------x---|` | 1/5 = 0.2


In fact, any relation can be used, as microtiming is less perceived on a pulse.
