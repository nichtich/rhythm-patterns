rhythms

This page allows to analyze and experiment with rhythmic patterns (aka rhythms).

A rhythm is a repeated sequence of beats and rests, for instance x--x--x-. See the collection of rhythms for more examples.

There are 2n possible rhythms in a sequence of n pulses, but some of these are more interesting then others.
