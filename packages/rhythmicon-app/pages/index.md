This page allows to analyze and experiment with rhythmic patterns (aka rhythms).

A rhythm is a repeated sequence of beats and rests, for instance `|x--x--x-|`.

There are 2<sup>n</sup> possible rhythms in a sequence of n pulses, but some of these are more interesting then others.
