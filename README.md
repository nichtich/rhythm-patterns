# rhythmicon

[![Test](https://github.com/nichtich/rhythmicon/actions/workflows/test.yml/badge.svg)](https://github.com/nichtich/rhythmicon/actions/workflows/test.yml)
[![NPM package name](https://img.shields.io/badge/npm-rhythmicon--rhythm-blue.svg)](https://www.npmjs.com/package/rhythmicon-rhythm)
[![NPM package name](https://img.shields.io/badge/npm-rhythmicon--vue-blue.svg)](https://www.npmjs.com/package/rhythmicon-vue)

The [Rhythmicon](https://en.wikipedia.org/wiki/Rhythmicon) one of the very first, short-lived electronic instruments. **rhythmicon** is a web application, a programming library, and a collection of rhythmic patterns:

- [rhythmicon-app](packages/rhythmicon-app#readme) is a web application to browse and analyze the collection of rhythms and other rhythmic patterns
- [rhythmicon-rhythm](packages/rhythmicon-rhythm#readme) is a JavaScript library to analyze and compute rhythmic patterns
- [rhythmicon-vue](packages/rhythmicon-vue#readme) is a JavaScript library of Vue components to display and interact with rhythmic patterns
- [rhythms](rhythms#readme) is a directory of rhythms, each described with their pattern and other properties
